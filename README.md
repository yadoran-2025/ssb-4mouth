# 🗣️ 사구동성 경제활동 게임

## 📌 개요
- 일상생활 속 경제 활동 예시를 통한 경제 3요소(생산, 소비, 분배) 학습 게임
- 웹 브라우저 기반 프레젠테이션용 소프트웨어
- 교사 주도하에 조별 대항전 형식으로 진행되는 참여형 수업 도구

---

## 🎯 학습 목표
- 현실의 다양한 경제 활동 사례를 살펴보고, 생산/분배/소비 중 어디에 해당하는지 정확히 분류함
- 조별 문제 풀이 과정을 통해 학생들 간의 협동심 및 순발력 배양

---

## 💻 게임 준비 (교사 세팅)
- **실행 방법**: 별도 설치 없이 `index.html` 파일을 웹 브라우저(크롬, 엣지 등)로 실행
- **디스플레이**: 교실 프로젝터 또는 대형 모니터(TV) 연결 후 전체화면 모드(`F11`) 권장
- **환경 설정 (메인 화면)**: 
  - 퀴즈 문제당 제한 시간(타이머) 설정 (기본 4초 권장)
  - 참가 조 이름 입력 및 세팅 (최대 8개 조 지원)

---

## 🧑‍🏫 수업 및 진행 가이드
### 1단계: 개념 학습 및 작전 타임
- 게임 시작 전, 칠판이나 자료를 활용해 생산, 소비, 분배에 대한 사전 교육 실시
- 조별로 정답을 맞히기 위한 작전 회의 및 복습 시간 부여

### 2단계: 조별 출전
- 학생들을 4인 1조로 구성
- 차례가 된 조(4명)가 교실 앞으로 나와 "도전!" 복창
- 진행자(교사)는 메인 화면에서 출전 조의 **[세트 X 시작]** 버튼 클릭

### 3단계: 사구동성(四口同聲) 퀴즈 진행
- 화면에 일상생활 문제 제시 및 타이머 작동 시작
- 4명의 조원은 타이머 종료 전 정답(생산/분배/소비 중 택1)을 **한목소리로 동시에(사구동성)** 외침
- 4명의 의견이 엇갈리거나, 시간 초과, 침묵 시 가차 없이 오답 처리

---

## 🎮 조작법 및 단축키
마우스 클릭 최소화 및 원활한 진행을 위해 직관적인 **키보드 단축키** 지원

- `Space` / `Enter` : **정답 확인** (정답 텍스트와 해설 표시)
- `O` (알파벳 오) : **정답 처리** (점수/콤보 누적 및 즉시 다음 문제로 이동)
- `X` : **오답 처리** (라이프 1개 차감되며 화면 점멸 효과 발생)
- `←` (좌방향키) : **이전 문제로 롤백** (진행자 조작 실수 보정용)

---

## 🏆 득점 시스템 및 우승
- **기회(라이프)**: 조별 기본 목숨 3개 부여, 모두 소진 시 즉시 게임 오버
- **콤보 점수**: 연속 정답 시 화면에 'COMBO!' 이펙트 발생 및 추가 보너스 점수 획득
- **동기 부여용 페널티 (메인 화면 점수판)**: 
  - 800점 이하: 점수칸 붉은색 폰트 및 붉은색 배경 음영 처리
  - 500점 이하: 불명예를 나타내는 **해골(💀)** 마크 추가 부여
- **최종 시상**: 전체 세트 종료 후 최고 점수 획득 조(1위)에 **왕관(👑)** 마크 토글. 우승팀에게 준비된 보상 지급 

---

## 🛠️ (선택) 문제 세트 커스텀 방법
1. `index.html` 파일을 텍스트 에디터(메모장, VS Code 등)로 오픈
2. 코드 내의 `const allQuizData = [ ... ]` 배열 파트 탐색
3. `{ q: "문제 내용", a: "정답", exp: "해설" }` 양식에 맞춰 문제 텍스트 변경
4. 저장 후 웹 브라우저 새로고침 시 즉시 반영됨
